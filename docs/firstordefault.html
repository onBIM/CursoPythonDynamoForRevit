FirstOrDefault

Definição

O método FirstOrDefault é utilizado para retornar o primeiro elemento de uma coleção ou o primeiro elemento que satisfaça uma condição.

Caso a coleção esteja vazia, ou não exista um elemento que satisfaça a condição, o método retorna None.

Exemplos

A forma mais simples de usar o método FirstOrDefault é sem passar nenhum argumento. Nesse caso, o método retorna o primeiro elemento da coleção.

import clr # Adiciona a refer√™ncia para o System.Core clr.AddReference("System.Core") # Importa os m√©todos de extens√£o do LINQ. import System clr.ImportExtensions(System.Linq) # cria uma lista de n√∫meros numeros = [1, 2, 3, 4, 5] # retorna o primeiro n√∫mero da lista numero = numeros.FirstOrDefault() OUT = numero # retorna 1

Vamos supor que tenhamos uma lista de pessoas e queremos retornar a primeira cujo nome comece com a letra "P".

import clr # Adiciona a refer√™ncia para o System.Core clr.AddReference("System.Core") # Importa os m√©todos de extens√£o do LINQ. import System clr.ImportExtensions(System.Linq) class Pessoa: def __init__(self, nome, idade): self.nome = nome self.idade = idade # cria uma lista de pessoas pessoas = [ Pessoa("Maria", 30), Pessoa("Jo√£o", 25), Pessoa("Jos√©", 20), Pessoa("Ana", 35), Pessoa("Pedro", 40), Pessoa("Carla", 45) ] # retorna a primeira pessoa que o nome comece com a letra "P" pessoa = pessoas.FirstOrDefault(lambda p: p.nome.startswith('P')) OUT = f"{pessoa.nome} - {pessoa.idade} anos"

Agora suponha que queremos a pessoa mais nova da lista.

# retorna a pessoa mais nova pessoa = \ pessoas \ .OrderBy(lambda p: p.idade) \ .FirstOrDefault()

Se quisermos retornar o último elemento da coleção, podemos usar o método LastOrDefault().

Seu funcionamento é similar ao FirstOrDefault, mas retorna o último elemento da coleção.
