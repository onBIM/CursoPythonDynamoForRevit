OrderBy

Definição

O método OrderBy é utilizado para ordenar os elementos de uma coleção.

Exemplo

Vamos supor que tenhamos uma lista de pessoas e queremos ordená-las pelo nome.

Usando o método OrderBy do Linq podemos fazer isso de forma muito mais eficiente e simples, sem a necessidade de usar um laço for.

import clr # Adiciona a refer√™ncia para o System.Core clr.AddReference("System.Core") # Importa os m√©todos de extens√£o do LINQ. import System clr.ImportExtensions(System.Linq) class Pessoa: def __init__(self, nome, idade): self.nome = nome self.idade = idade # cria uma lista de pessoas pessoas = [ Pessoa("Maria", 30), Pessoa("Jo√£o", 25), Pessoa("Jos√©", 20), Pessoa("Ana", 35), Pessoa("Pedro", 40), Pessoa("Carla", 45) ] # ordena as pessoas pelo nome pessoas_ordenadas = pessoas.OrderBy(lambda p: p.nome) OUT = pessoas_ordenadas.Select(lambda p: f"{p.nome} - {p.idade} anos")

Se quisermos ordenar as pessoas pelo nome de forma decrescente, podemos usar o método OrderByDescending.

# ordena as pessoas pelo nome de forma decrescente pessoas_ordenadas = pessoas.OrderByDescending(lambda p: p.nome)
