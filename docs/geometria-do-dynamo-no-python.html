Geometria do Dynamo no Python

Podemos utilizar a biblioteca de geometria do Dynamo no Node Python Script para fazermos operações geométricas.

Veja como importar a biblioteca de geometria do Dynamo no Python:

import clr clr.AddReference('ProtoGeometry') from Autodesk.DesignScript import Geometry as DynamoGeometry

Agora podemos utilizar as funções de geometria do Dynamo no Python.

import clr clr.AddReference('ProtoGeometry') from Autodesk.DesignScript import Geometry as DynamoGeometry # obtÃ©m os valores das coordenadas Ã  partir dos inputs # do node Python Script no Dynamo x = IN[0] y = IN[1] z = IN[2] # Criando um ponto ponto = DynamoGeometry.Point.ByCoordinates(x, y, z) # Output do node Python Script no Dynamo OUT = ponto

Dessa forma, podemos usar todos os recursos de geometria do Dynamo no Python.

Somente as geometrias que você atribuir na variável OUT serão exibidas no Dynamo.

Isso é uma grande vantagem, pois significa que as geometrias utilizadas internamente no código Python não serão exibidas no Dynamo, o que trará granhos de performace para a sua rotina.

Por outo lado é um ponto de atenção para o desenvolvedor, pois já que as geometrias não são exibidas no Dynamo, é preciso abstrair mais o raciocínio para fazer o que você deseja. E também pode trazer dificuldades para debugar.

As principais recomendações são:

  1. Evite códigos grandes demais em um único node Python Script no Dynamo.

  2. Divida o código em funções.

  3. Faça funções pequenas e com um único propósito.

  4. Utilize comentários para explicar o que o código faz.

  5. Utilize variáveis com nomes significativos.

  6. Utilize a função print() para debugar o código se necessário.
